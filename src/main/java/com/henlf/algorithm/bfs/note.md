## 1. 为什么 BFS 可以找到最短距离，DFS 不行吗？
> BFS 的逻辑，depth每增加一次，队列中的所有节点都向前迈一步，这保证了第一次到达终点的时候，走的步数是最少的。
> DFS 不能找最短路径吗？其实也是可以的，但是时间复杂度相对高很多。
> BFS 借助队列做到一次一步「齐头并进」，是可以在不遍历完整棵树的条件下找到最短距离的。
  形象点说，DFS 是线，BFS 是面；DFS 是单打独斗，BFS 是集体行动
